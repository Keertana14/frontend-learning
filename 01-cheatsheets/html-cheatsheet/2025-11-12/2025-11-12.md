### ğŸ·ï¸ Tag Roles

**<!DOCTYPE html>:**
This line flags to the browser that this is an HTML5 file. DOCTYPE is used to specify the version of the HTML file the code is written in. This is not an HTML element.
**HTML tag: <html> </html>:**
The html element is the root element that wraps all other elements in it.
**HEAD tag: <head> </head>:**
The head element contains the key information of the document that does not appear on the web page.
**TITLE tag: <title> </title>**
The title element contains the title of the web page; this is not the file name.
**BODY tag: <body> </body>**
The body element will contain all the visible content on the web page.
**lang attribute** - specifies the primary language of the document
**meta tags** - provides important information to the browser, we do not use them while writing html, they are more commonly grabbed from other places.
**charset = UTF-8** - This is a system for computers to store numbers, letters symbols and special characters.
